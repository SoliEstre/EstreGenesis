Install the plugin

Constellation ships as a Claude Code plugin with embedded MCP server.

# Add the EstreGenesis marketplace:
/plugin marketplace add SoliEstre/EstreGenesis

# Install the constellation plugin:
/plugin install constellation@estregenesis-plugins

# Configure env (one of):
CONSTELLATION_WS_URL=ws://localhost:7878/ws
CONSTELLATION_TOKEN=...    # OR CONSTELLATION_UPSTREAM_KEY / CONSTELLATION_COLLAB_KEY
CONSTELLATION_STATE_PATH=/path/to/state.json
